# FinApp2P
App de controle financeiro e patrimonial, mobile-first, em TypeScript com arquitetura leve (frontend + backend + shared).

## Estrutura
- `frontend`: Vite + TypeScript (vanilla), telas mobile-first.
- `backend`: Node + TypeScript, HTTP simples.
- `shared`: tipos utilitários compartilhados.

## Comandos
- `npm run dev:frontend` — inicia o frontend em modo dev.
- `npm run dev:backend` — inicia o backend com reload via tsx.
- `npm run build` — build de frontend e backend.

Requisitos: Node >= 20 e npm com workspaces habilitados.
